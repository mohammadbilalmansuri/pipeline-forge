# PipelineForge

A modern visual pipeline builder for creating, connecting, and validating workflow graphs with ease.

## Features

- **Drag & Drop Interface** - Intuitive node-based workflow creation
- **10 Node Types** - Input, Output, Text, LLM, API, Code, Knowledge Base, Note, Delay, Transform
- **Variable System** - Dynamic variable detection with `{{variable}}` syntax
- **Real-time Validation** - DAG (Directed Acyclic Graph) validation
- **Persistent Storage** - Auto-save to local storage
- **Responsive Design** - Works on desktop and tablet devices

## Tech Stack

- JavaScript
- React with Vite
- ReactFlow
- Zustand
- Tailwind CSS

## Setup

1. Install dependencies: `pnpm install`
2. Run the development server: `pnpm run dev`
